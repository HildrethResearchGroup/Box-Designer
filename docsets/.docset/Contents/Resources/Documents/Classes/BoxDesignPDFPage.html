BoxDesignPDFPage

class BoxDesignPDFPage : PDFPage

This class is the main driver for creating and drawing a single PDF page of the box model walls.

Authors

CSM Field Session Summer 2020, Fall 2020, and Dr. Owen Hildreth.

Note

BoxDesignPDFPage.swift was created on 5/27/2020. Additionally, curved paths are not able to be drawn with how this code is written – the NSBezierPath.ElementType.curveTo is just drawn like a line right now.
